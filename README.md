# Zoo

Created a project zoo with HTML files.
A simple and interactive Zoo-themed webpage built using HTML, showcasing different animals with images, descriptions, and organized sections.
This project demonstrates the fundamentals of HTML structure, semantic tags, and basic web layout design.

Features:
1)  Organized animal sections with headings and images
2)  Informative text about each animal
3) Clean and beginner-friendly HTML code

Purpose:
              Created as a beginner web development project to practice webpage creation and layout structuring using HTML.
